# Регламент ЧМ-2026 (справочник для логики турнира)

> Источники: [Wikipedia — 2026 FIFA World Cup](https://en.wikipedia.org/wiki/2026_FIFA_World_Cup),
> [Wikipedia — knockout stage](https://en.wikipedia.org/wiki/2026_FIFA_World_Cup_knockout_stage),
> [ESPN — format & tiebreakers](https://www.espn.com/soccer/story/_/id/47108758/2026-fifa-world-cup-format-tiebreakers-fixtures-schedule).
> Проверено 05.06.2026.

## Общий формат
- **48 команд**, **12 групп** (A–L) по 4 команды.
- Каждая команда играет 3 матча в группе.
- В плей-офф выходят: **1-е и 2-е места** всех групп (24) + **8 лучших третьих мест** → **Round of 32**.
- Плей-офф (single-elimination): R32 → R16 → 1/4 → 1/2 → **матч за 3-е место** → финал.
- Всего 104 матча (было 64). Финал — MetLife Stadium.

## Тай-брейки в группе (порядок)
1. Очки
2. Разница мячей
3. Забитые мячи
4. (при равенстве — мини-турнир между равными: очки → разница → голы в личных встречах)
5. Команд. fair-play (дисциплинарный балл)
6. Позиция в рейтинге FIFA
> ⚠️ Точный порядок п.4 (личные встречи vs общий fair-play) уточнить по официальному
> регламенту FIFA при реализации Фазы 2. В нашем боте дисциплинарный балл взять не из чего
> (нет карточек в прогнозе) → тай-брейк сведём к: очки → разница → голы → (личные встречи) →
> рейтинг/жребий. Решение зафиксировать в коде сервиса.

## Отбор 8 лучших третьих мест
Ранжирование третьих мест: **очки → разница мячей → забитые → fair-play → рейтинг FIFA**.
Берутся 8 лучших из 12.

⚠️ **Ключевая сложность:** к каким именно матчам R32 привязываются 8 третьих мест —
зависит от того, из каких групп эти 8 команд. FIFA использует предопределённую таблицу
на **495 комбинаций** (C(12,8) = 495 вариантов выбора 8 групп из 12). Эту таблицу нужно
будет загрузить как справочник при реализации Фазы 3.

## Что это значит для нашей реализации
- Структуру сетки (матчи 73–102, связи «победитель/проигравший матча N») храним как
  справочник `bracket_slots` — она статична.
- Привязка 8 третьих мест к слотам R32 — отдельная таблица-справочник (495 комбинаций),
  загружается в Фазе 3.
- В плей-офф нужен победитель каждого матча → в прогнозе ничьих быть не может
  (если счёт равный — нужен механизм «кто прошёл»: доп. вопрос/пенальти). Решить в Фазе 3 UX.
