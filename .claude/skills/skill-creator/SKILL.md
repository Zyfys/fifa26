# Skill Creator

Помогает создавать новые скилы в `.claude/skills/` для проекта «Прогноз на ЧМ-2026».

## Когда использовать
- Пользователь явно просит создать новый скил.
- Есть повторяющийся процесс, который стоит оформить как переиспользуемый навык.

## Как создать скил

1. **Уточни задачу.** Что скил должен делать, когда активироваться, какие шаги выполняет.
2. **Придумай имя** в kebab-case (например `generate-bracket-pdf`).
3. **Создай папку** `.claude/skills/<имя>/` и файл `SKILL.md` внутри.
4. **Заполни frontmatter:**
   ```yaml
   ---
   name: <имя-в-kebab-case>
   description: <когда активируется и что делает — от этого зависит автоподхват>
   ---
   ```
5. **Опиши тело:** назначение, когда использовать, пошаговая инструкция, правила/ограничения.
6. При необходимости добавь вспомогательные файлы (скрипты, шаблоны) в ту же папку и сошлись на них.

## Правила оформления
- `description` — самое важное: по нему скил подбирается автоматически. Пиши конкретно,
  с триггерами («когда…»).
- Один скил — одна понятная зона ответственности.
- Инструкции — короткие, по шагам, без воды.
- Не дублируй существующие скилы — сначала проверь `.claude/skills/`.

## Шаблон нового SKILL.md
```markdown
---
name: example-skill
description: Что делает и когда активируется (триггеры).
---

# Example Skill

## Когда использовать
- ...

## Шаги
1. ...
2. ...

## Правила
- ...
```
