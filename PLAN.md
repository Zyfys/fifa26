# План проекта: Прогноз на ЧМ-2026 (Bracket World Cup 2026)

## Решения (зафиксированы на этапе планирования)
- **MVP** = полный цикл прогноза: группы → таблицы → плей-офф → финал →
  индивидуальные награды → символическая сборная → красивый PDF.
- **Главные риски**: полная база игроков (позиции/возраст/клуб); логика плей-офф +
  отбор 8 лучших третьих мест; PDF-вёрстка; UX заполнения 100+ матчей и выборов.
- **Старт** = данные + схема БД (фундамент для всего).

## Целевой формат турнира (ЧМ-2026)
- 48 команд, 12 групп по 4.
- Из группы выходят 1-е и 2-е места (24) + 8 лучших третьих мест → стадия 1/16.
- Плей-офф: 1/16 → 1/8 → 1/4 → 1/2 → **матч за 3-е место** → финал.
- После каждого этапа победители автоматически переходят в следующий раунд.

## Полный объём прогноза (из спека)
1. **Групповой этап** — ввод счёта всех матчей каждой группы; автоматический расчёт
   таблицы (очки, В/Н/П, забито/пропущено, разница) и определение выходящих.
2. **Плей-офф** — автоформирование сетки, ввод счёта, автоопределение победителей.
3. **Индивидуальные награды** (выбор только из списков БД):
   - 🏆 Чемпион, 🥈 финалист, 🥉 3-е место
   - ⚽ Золотая бутса (бомбардир) + прогноз числа голов
   - ⭐ Золотой мяч (лучший игрок)
   - 🌟 Молодой игрок (фильтр по возрасту)
   - 🧤 Золотая перчатка (только вратари)
   - 🚀 Открытие турнира (игрок)
   - 🔥 Сборная-сенсация, 📉 Команда-разочарование
4. **Символическая сборная 4-3-3** — ВРТ×1, ЗАЩ×4, ПЗ×3, НАП×3, фильтр по позициям.
5. **PDF** — визуальный футбольный отчёт со всем перечисленным.

## Модель данных (черновик)
**Справочники (сид):**
- `teams` — 48 сборных
- `groups` — 12 групп
- `players` — игроки: имя, `position` (GK/DF/MF/FW), `birth_date`/возраст, `team_id`
- `group_matches` — расписание групповых матчей (слоты)
- `bracket_slots` — структура сетки плей-офф

**Пользовательские данные:**
- `users` — telegram_id, username, прогресс, дата создания
- `predictions_group` — счёт по групповым матчам
- `predictions_bracket` — счёт по матчам плей-офф
- `predictions_awards` — индивидуальные награды (FK на teams/players)
- `predictions_team_of_tournament` — символическая сборная (FK на players + позиция)
